## 🚀 Featured Projects

### 🌦️ [WeatherApp](https://github.com/XTejeshX/WeatherApp)
> Real-time desktop weather app with a modern dark-themed GUI

Built with **Python + PyQt5**, this app fetches live weather from the OpenWeatherMap API with full error handling for network, HTTP, and timeout failures. Supports real-time °C/°F toggling and uses emoji-based visual condition codes.

**What it shows:** API integration · GUI development · Error handling · OOP with Python

`Python` `PyQt5` `REST API` `OpenWeatherMap`

---

### ⚙️ [dataset_cli](https://github.com/XTejeshX/dataset_cli)
> A modular CLI tool for downloading and preprocessing datasets

Structured across three clean modules — `downloader.py`, `processor.py`, and `main.py` — this tool fetches any CSV from an HTTP(S) endpoint and runs a preprocessing pipeline in one command. Built with good separation of concerns from the start.

**What it shows:** CLI design · Modular Python architecture · Data pipeline thinking · Pandas

`Python` `Pandas` `Requests` `CLI`
